Chaotic-AUR

Automated building repo for AUR packages

Disclaimer:

This repo is generated on-demand from packages I use on my personal computer.
If you find any package license infringement, please notify me at root[at]pedrohlc.com, so I can remove and blacklist it.

About:

Most packages available in this repo are automatically built from the AUR sources.

  • retdec-git, vlc-git, wireguard-dkms-git, rpcs3-git, ffmpeg-full, firefox-wayland-hg, llvm-*-git are building in the morning (4 am at America/Sao_Paulo).
  • wine-tkg-git variants are building in the sunset (7 pm at America/Sao_Paulo).
  • Other packages are building hourly.
  • Every package is building in a clean chroot.
  • This server is hosted at UFSCar's datacenter.
  • @BangL provides a germany mirror at http://chaotic.bangl.de
  • @LordKitsuna provides an american mirror at https://repo.kitsuna.net

Feel free to create an issue for suggestions, flagging database corruptions and everything else: Issues
Build bugs should be reported to source package's maintainers.

How to use it?

Add to /etc/pacman.conf:

[chaotic-aur]
Server = http://lonewolf-builder.duckdns.org/$repo/x86_64
Server = http://chaotic.bangl.de/$repo/x86_64
Server = https://repo.kitsuna.net/x86_64

For signature check, add my key:

sudo pacman-key --keyserver keys.mozilla.org -r 3056513887B78AEB
sudo pacman-key --lsign-key 3056513887B78AEB

Recommendations:

  • If you're using other third parties repositories, I recommend you to put chaotic-aur before them on /etc/pacman.conf. We update most packages hourly and follow AUR's PKGBUILDs, that makes this repository more suitable to be the first option when downloading packages.
  • If you need to omit one package update to use locally built ones (e.g: with native optimizations), you can add 'IgnorePkg = ...' to repo's section in /etc/pacman.conf
  • You can pick the repository from which you download a package, like this: pacman -S chaotic-aur/dxvk-mingw-git. Some AUR's helpers support it too, e.g: yay -S aur/firefox-hg
  • You may want to use powerpill, so you download from all mirrors simultaneously. I recommend doing updates this way: sudo pacman -Sy && sudo powerpill -Su && yay -Su
